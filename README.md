
# JavaScript Console Quiz Game

A simple interactive quiz game built with JavaScript that runs directly in the web browser console. This project demonstrates the use of arrays, loops, functions, and conditional logic.

## How to Play
1. Open your web browser (Chrome, Firefox, Edge, etc.).
2. Open the **Developer Tools** by pressing `F12` or right-clicking the page and selecting **Inspect**.
3. Click on the **Console** tab.
4. Copy the code from `quiz.js`.
5. Paste the code into the console and press **Enter**.
6. Answer the questions in the popup prompts.

## Features
* **Multiple Questions:** loops through an array of question objects.
* **Input Validation:** handles different capitalizations and extra spaces using `toLowerCase()` and `trim()`.
* **Immediate Feedback:** tells you if you are correct or wrong immediately after answering.
* **Score Tracking:** displays your final score at the end of the quiz.

## Questions Included
1. Geography (Capital of France)
2. Astronomy (Red Planet)
3. Math (Simple Addition)
4. Politics (Current President)
5. Technology (Web Development Language)
